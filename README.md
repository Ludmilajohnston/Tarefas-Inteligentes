# Tarefas Inteligentes

Aplicativo de gerenciamento de tarefas desenvolvido para a disciplina de Programação para Dispositivos Móveis.

---

#  Sobre o projeto

O **Tarefas Inteligentes** é um aplicativo PWA (Progressive Web App) desenvolvido para auxiliar na organização de tarefas do dia a dia.

A aplicação permite criar, editar, concluir e excluir tarefas, além de organizar atividades por categorias e prioridades.

---

#  Funcionalidades

##  Gerenciamento de tarefas

* Adicionar tarefas
* Editar tarefas
* Excluir tarefas
* Concluir tarefas

---

##  Categorias

As tarefas podem ser organizadas em:

* 📚 Estudos
* 💼 Trabalho
* 🏠 Pessoal

---

##  Sistema de prioridades

As tarefas possuem níveis de prioridade:

* 🔴 Alta
* 🟡 Média
* 🟢 Baixa

Com destaque visual através de cores nos cards.

---

##  Filtros

Visualização de:

* todas as tarefas;
* tarefas pendentes;
* tarefas concluídas.

---

##  Dark Mode

Modo escuro com salvamento automático da preferência do usuário.

---

##  Contador de tarefas

Exibição da quantidade de tarefas pendentes em tempo real.

---

##  Ordenação automática

As tarefas são organizadas automaticamente por:

1. status;
2. prioridade.

---

##  PWA (Progressive Web App)

O aplicativo pode ser:

* instalado no celular;
* utilizado offline;
* executado como aplicativo nativo.

---

#  Funcionamento Offline

O sistema utiliza Service Worker para armazenar arquivos em cache e permitir funcionamento mesmo sem conexão com internet.


---

#  Deploy

O projeto foi publicado utilizando GitHub Pages para permitir acesso online e instalação em dispositivos móveis.

---

#  Como executar o projeto

1. Baixe ou clone o repositório
2. Abra a pasta do projeto
3. Execute com Live Server
4. Acesse pelo navegador

---

#  Instalação no celular

1. Abra o link da aplicação no navegador
2. Clique em:

   * “Adicionar à tela inicial”
   * ou “Instalar aplicativo”
3. O aplicativo será instalado no dispositivo
